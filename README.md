# CuoiKy-Chat

Mô tả
- Dự án: Ứng dụng chat nhóm cho đồ án/cuối kỳ.
- Mục tiêu: Demo chat thời thực, quản lý người dùng, lưu trữ lịch sử.

Hướng dẫn nhanh
1. Cài đặt phụ thuộc (tùy stack):
	- Node: `npm install` hoặc `yarn install`
	- Python: `pip install -r requirements.txt`
2. Chạy ứng dụng:
	- Node: `npm start` hoặc `yarn start`
	- Python: `python app.py`

Ghi chú
- Lưu các khóa/biến môi trường vào file `.env` và không commit.
- Xem `README_DEV.md` để biết hướng dẫn cấu hình môi trường phát triển.
